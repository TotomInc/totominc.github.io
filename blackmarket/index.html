Where I am in the development

So, the game has currently nice core features, who may keep you a good time. But I know, there are a lot of features that I said (on reddit) that there is not : like prestige upgrades, achievements, safe/work mode, ...
For now, I put the development of the game paused.
I don't have any great idea of things to implement and I'm a bit discouraged of adding more content.
A big thanks for you, for playing my game, you were a total of approximatively 15'000 unique players/visitors and something like 1'000 regular players to go everyday on the game.
This is not nothing, it's huge for my first real project and makes me want to continue in programming. Thanks you! Maybe I will go back to the development of the game , so take a look from time to time! :)

  • v0.1 : beta release!

    Beta release of the game!
    - Code has been fully rewritten, with some optimizations!
    - New fresh clean UI (essentially build on Bootstrap).
    - Added ranks, which give a total cash multiplier.
    - Helpers are removed.

    v0.11 : UI changes!

    - The game UI have changed, I'm always working on it, especially on the upgrades tab.
    - Changed some prices of builds/drugs, game get a good balancing according to me.
    - Fixed saves bug when I add more content like new builds/dealers. Please, let me know if there is still saving bugs!

    v0.12 : Content update!

    - Cocaine builds and dealers have been added!
    - 3 upgrades added (cocaine price) and some of them have been rebalanced.
    - When you bought an upgrade, it become green instead of grey.
    - Fixed some bugs.
    - Changed some things about the background, navbar and actions panel colors.

    v0.13 : prestige update!

    - Added a prestige system : you got a prestige rank and experience. Experience will determine your prestige rank which will give you a "prestige multipier" on every cash income! (Shooting action and drugs for example)

    v0.14 : rebalancing the game

    - Builds/dealers have been totally rebalanced!
    - Drug prices have been a bit boosted.

    v0.15 : always balancing the game

    - New upgrades have been added (1 more for weed/meth/cocaine).
    - Builds/dealers have been rebalanced (again!).
    - Drug prices have been boosted, especially meth and cocaine.
    - Experience is now 2x easier to gain!
    - Code have been a bit optimized.
